## TL;DR

Obsidian is AppSec-approved at Amazon, available via IT Marketplace, used by 4000+ Amazonians — but Obsidian Sync is not allowed for work vaults.

## Details

**Availability:**
- AppSec approved since 2021 (under review again in 2025)
- Licenses via IT Marketplace — most auto-approved within 1-2 days
- 14-day trial without a license
- No Obsidian account needed for the commercial license
- Community: `#obsidian-interest` on Slack (4000+ members)

**Key constraint:** Obsidian Sync (their paid cloud service) cannot be used for vaults containing Amazon work content. This would upload confidential data to a third-party cloud, violating data handling policies. Personal vaults with no Amazon data can use Sync.

**Amazon-specific plugins:**
- Internal Plugin Marketplace
- PhoneTool (employee directory queries)
- Amazon Meetings (daily schedule integration)
- Midway (auth plugin, not fully supported)

These require developer access via code.amazon.com and mwinit for Midway auth.

**Resources:**
- Wiki: `w.amazon.com/bin/view/Obsidian.md/`
- Slack: `#obsidian-interest`

## When This Matters

When considering note-taking tools at Amazon, or when evaluating whether Obsidian can be used for a specific workflow involving Amazon data.
